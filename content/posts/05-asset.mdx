### 1. 什么是资产？

同样的东西，对于不同的人来说，所看到的价值是不一样，对应的所反映的价格也就千差万别。

同样的概念，对于不同来说，思考的逻辑也是不一样的，最后塑造不一样的观念，驱动不一样的行动，最后获得不一样的结果。



从这里我们思考一个简单的东西，**资产**，资产到底是什么。



很多人以为，钱包里面的存款，手上有的房子车子是资产，但我思考了一下

* 如果钱包里面的存款不能获得收益
* 如果手上的房子不能获得租金，或者二级市场卖出去
* 如果车子只是放在地下车库，从来不开放

那么这些东西很难形成自己的资产。



那什么能形成自己的资产呢

* 钱包里面的钱放在存储中获得年化(月化)收益
* 把房子出租出去，每个月收租金
* 车子每天开，完成他的使用以及工具的作用，从而间接的节省时间

**资产能持续为我们带来现金流的东西**

一栋别墅，花了很高的价格买到，但是想出售(出租)，很难找到买家，这就不算我们拥有的资产。

### 2. 资产定价模型

要理解资产的价值，需要将未来现金流折现

> * 一个刚刚大学毕业的男生，随着社会工作的阅历越来越多，从打工到创业，积累产品
> * 一个刚刚毕业的大学女生，美貌资产在这个时候是最高的，而随着时间流逝，美貌也在不断地变低



本质上，人的工作/企业/产品，都是一份资产，能够为自己源源不断的带来现金流。

```latex
资产价值 = Σ (未来现金流 / (1 + r)^t)
```

一个资产能够带来多少钱，或者他的价值到底是多少，则是当下和未来他的所有可吸引的现金流总和。



而决定资产定价模型的几个要素大概有

> * 现金流大小
> * 持续时间
> * 确定性风险大小



如果每个月都在努力的工作上班，那么每个月的现金流都是很固定的（可能会有几次加薪）而且这个持续时间大概是40年，风险很小( 但其实很大，行业变化很快 )



如果按照商业的模式去进行思考，其实每个月的现金流都是很不固定的，可能很久很长一段时间都养不起员工(那么就得裁员)，也可能市场行情很好， 一次机会赚到很多钱， 但风险其实也是很大的。



本身我们的时间只有一份，所以就是选择，按照自己能够承担的方式，按照自己能够接受的风险去选择，未来就是由这些很多很多的小的选择去组成一个大的结果。

如果我把这些所有的事情都亲力亲为，长期来看，这家企业一定是低效臃肿的

### 4. 销售结果

**我改变了我的思维模式。**

我之前的专注与细节上的迭代和优化，我现在把我的视角切成了，结果导向。

一份资产，觉得他的价值，是有共识决定的，共识则由大家看到的结果来决定的。

> * BTC (2100万不可新增， POW算力挖矿)
> * 对外不应该靠时间付费， 这个是无底洞，更重要的是应该靠结果付费



提升价值永远的第一位的，而更重要的是，要永远在最短的时间内创造更多的价值。

**结果是可以批量销售的。**

之前的思维，让我们可以训练为，做更好的实现，写出更优化的方案，但是这个思路有一个问题，就是我们一直在优化的是这个过程。

而另外一个很多打工人并没有注意到的点是

> 市场买单的永远是结果，而非过程。



* 市场买单的是一个手机， 能打电话能拍照，并不关心租房师为他花了多少时间
* 市场买单的是，这个方案能够带给他多少增量，提高多少仓位收益，而非写出这个策略花了多少时间
* 市场买单的是，BTC的稀缺，共识，去中心化，网络效应， 而不会为中本聪到底是谁？花了多少时间去编写这套程序买单。



**我应该售卖结果而非售卖自己的时间。**

用一定的时间去达到资产，资产的结果进行批量化出售，才是这篇文章最核心的逻辑，也是我最核心的观念上的转变

我觉得很多人都听过这句话，巴菲特老爷子说的。

事的增长率远远大于钱本身。
